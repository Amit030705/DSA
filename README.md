# OOP in C++

Welcome to **OOP-in-CPP**! 🚀

This repository is a collection of well-commented C++ programs designed to help beginners and intermediate programmers understand Object-Oriented Programming concepts from scratch.

## Topics Covered

- Classes and Objects
- Constructors and Destructors
- Encapsulation
- Inheritance
- Polymorphism
- Abstraction
- Friend Functions
- Virtual Functions
- Operator Overloading
- Static Members
- Templates
- Exception Handling
- File Handling
- And many more...

Each program is written with readability and learning in mind, making this repository a useful resource for students, coding enthusiasts, and anyone looking to strengthen their C++ OOP fundamentals.

⭐ Feel free to explore, fork, and contribute!
